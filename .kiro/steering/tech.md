# Technology Stack

## Core Technologies

- **Framework**: React 19.2.0 with TypeScript
- **Build Tool**: Vite 7.2.4
- **Routing**: React Router DOM 7.9.6
- **Styling**: Tailwind CSS 4.1.17
- **Icons**: Lucide React 0.555.0

## Development Tools

- **TypeScript**: 5.9.3
- **ESLint**: 9.39.1 with TypeScript ESLint, React Hooks, and React Refresh plugins
- **PostCSS**: For Tailwind CSS processing

## Common Commands

```bash
# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview

# Run linter
npm run lint
```

## Build Configuration

- Vite with React plugin for fast HMR
- TypeScript with strict type checking
- ESLint with recommended configs for React and TypeScript
- Tailwind CSS configured to scan all source files

## Project Type

ES Module based project (type: "module" in package.json)
